## Quick start

```bash
# Clone and setup
git clone https://github.com/sciexp/typescript-nix-template.git
cd typescript-nix-template
nix develop
bun install

# Start development
just dev
```

See [Getting started guide](/guides/getting-started/) for complete setup instructions.

## Using as a template

This repository serves dual purposes:

1. **Working deployment** - Live example at ts-nix.scientistexperience.net
2. **Forkable template** - Starting point for your TypeScript projects

See [Template usage guide](/guides/template-usage/) for customization instructions.
